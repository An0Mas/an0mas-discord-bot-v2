# 言語設定
すべての出力について、以下の言語ルールに従ってください：
1. **主要言語**: すべての回答は**日本語**で行ってください。
2. **成果物**: すべての成果物（プラン、ウォークスルーなど）は日本語で記述してください。
3. **コードコメント**: コードコメントは日本語で記述してください。

# 実装ルール
- このチャット（Antigravity）は主に**相談・仕様書作成用**として使用
- 実際のコード実装が必要な場合は、**実装前に必ず確認を取る**こと
- 実装は主に**Codex**で行う

# 実装時のドキュメント編集
実装を行う際は、以下のドキュメントの編集・追記が必要：

| ファイル | 内容 | 編集タイミング |
|---------|------|---------------|
| `docs/SCOPE.md` | プロジェクトのスコープ定義 | 機能追加・変更時 |
| `docs/SPEC.md` | 技術仕様書 | 機能追加・変更時 |
| `docs/COMMAND.md` | コマンド一覧 | コマンド追加・変更時 |
| `docs/HELP.md` | ユーザー向けヘルプ | コマンド追加・変更時 |
| `docs/DETAILS/*.md` | 各コマンドの詳細仕様 | 該当コマンド変更時 |
| `docs/PLANS.md` | 開発計画・ロードマップ | 計画変更時 |
