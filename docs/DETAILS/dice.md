# /dice

ダイスロールコマンド。

## 基本情報

| 項目     | 内容                 |
| -------- | -------------------- |
| コマンド | `/dice [expression]` |
| 権限種別 | Everyone             |
| 使用場所 | サーバー内           |

## 引数

| 引数         | 必須 | 説明                                                    |
| ------------ | :--: | ------------------------------------------------------- |
| `expression` |  -   | NdM形式のダイス式（例: `2d6`, `1d100`）。省略時は `1d6` |

### NdM形式

- `N`: ダイスの個数（1〜100）
- `d`: 区切り文字
- `M`: ダイスの面数（2〜1000）

例:

- `1d6` → 6面ダイスを1回
- `2d6` → 6面ダイスを2回振って合計
- `1d100` → 100面ダイスを1回
- `3d20` → 20面ダイスを3回振って合計

## 出力例

### 単一ダイス

```
🎲 @User rolled 1d6
Result: 4
```

### 複数ダイス

```
🎲 @User rolled 2d6
Result: 3 + 5 = 8
```

### 大量ダイス

```
🎲 @User rolled 10d6
Result: 3 + 5 + 2 + 6 + 1 + 4 + 6 + 2 + 3 + 5 = 37
```

## エラーケース

| ケース         | メッセージ                          |
| -------------- | ----------------------------------- |
| 無効な形式     | `❌ 無効な形式です。例: 2d6, 1d100` |
| 個数が多すぎる | `❌ ダイスの個数は1〜100個までです` |
| 面数が多すぎる | `❌ ダイスの面数は2〜1000までです`  |

## 実装メモ

- 乱数生成: `Math.random()` を使用
- 結果は公開表示（ephemeral: false）
- Embed形式ではなく、シンプルなテキスト返信
