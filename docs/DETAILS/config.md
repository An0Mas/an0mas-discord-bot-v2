# /config — 設定確認（オーナー専用）

## 概要

Botの設定・権限設定を確認するオーナー専用コマンド。

## サブコマンド

### /config show

現在のサーバーの設定を表示する。

### /config permissions <コマンド>

指定コマンドの許可ユーザー・許可ロール一覧を表示する。

## 引数

| 引数     | 説明                   |
| -------- | ---------------------- |
| コマンド | 対象コマンド名（任意） |

## 表示内容

### /config show

- Guild ID
- Bot利用許可状態（有効/無効）

### /config permissions

- 許可ユーザー一覧
- 許可ロール一覧

## 備考

- Botオーナー（.envのOWNER_ID）のみ実行可能
- Ephemeralで表示（本人のみ）
