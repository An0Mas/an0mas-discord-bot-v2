# /mention-reactors

## 概要

特定のメッセージにリアクションを押した人全員にメンションを送ります。

## 使い方

```
/mention-reactors message:<メッセージIDまたはリンク>
```

## 引数

- `message`：対象メッセージのIDまたはリンク（必須）

## 動作フロー

1. コマンドを実行するとリアクション一覧がボタンで表示されます
2. メンションを送りたいリアクションのボタンを押します
3. そのリアクションを押した人全員にメンションが送信されます

## ボタン

- `[絵文字 N人]`：特定のリアクションを押した人にメンション
- `[📋 全員]`：すべてのリアクションを押した人にメンション（重複排除）

## 権限

- **Botオーナー** または `/allow` で許可されたユーザー/ロールのみ

## 注意事項

- **同一サーバー内のみ対象可能**（別サーバーのリンクは使用不可）
- メッセージリンクを使用すれば同一サーバー内の別チャンネルのメッセージも対象可能
- 100人を超えるリアクションは最初の100人のみ取得
- メンション文が長い場合は複数メッセージに分割送信

## 例

```
/mention-reactors message:1234567890123456789
/mention-reactors message:https://discord.com/channels/123/456/789
```

## メッセージリンクの取得方法

対象メッセージを右クリック →「メッセージリンクをコピー」を選択します。
