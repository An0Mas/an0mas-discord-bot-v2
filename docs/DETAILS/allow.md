# /allow — 権限管理（オーナー専用）

## 概要
Botの利用許可・コマンド権限を管理するオーナー専用コマンド。

## サブコマンド

### /allow guild add
現在のサーバーでBotの利用を許可する。

### /allow guild remove
現在のサーバーでBotの利用を停止する。

### /allow user add <コマンド> <ユーザー>
指定ユーザーに特定コマンドの実行権限を付与する。

### /allow user remove <コマンド> <ユーザー>
指定ユーザーから特定コマンドの実行権限を削除する。

### /allow role add <コマンド> <ロール>
指定ロールを持つメンバーに特定コマンドの実行権限を付与する。

### /allow role remove <コマンド> <ロール>
指定ロールから特定コマンドの実行権限を削除する。

## 引数
| 引数 | 説明 |
|------|------|
| コマンド | 対象コマンド名（例: bosyu, remind） |
| ユーザー | 対象ユーザー |
| ロール | 対象ロール |

## 備考
- Botオーナー（.envのOWNER_ID）のみ実行可能
- Guild許可チェックより先に実行される（未許可Guildでも使用可能）
