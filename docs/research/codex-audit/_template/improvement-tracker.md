## 運用ルール

- 着手前に `参照（根拠ファイル）` と `plan-10` / `risks` の該当項目を確認してから作業する。
- 実施者は `AI` または `自分` を記載する。
- 着手時に `⬜ -> 🔄`、完了時に `🔄 -> ✅` へ更新する。
- 保留時は理由と再開条件をメモに残す。
- `参照（根拠ファイル）` に、実装対象や根拠となるファイルを必ず記載する。
- 完了時は最低限以下を記録する。

  - 必要な手動/自動テスト結果
